rune Documentation

This is the documentation section for rune, the automatic differentiation and JIT compilation library.

Getting Started

rune provides JAX-inspired automatic differentiation and JIT compilation using OCaml 5 algebraic effects.

Quick Example

open Rune

(* Define a function *)
let f x = Tensor.(x * x + sin x)

(* Get gradient function *)
let grad_f = grad f

(* JIT compile for performance *)
let jit_f = jit f

(* Compose transformations *)
let fast_grad_f = grad (jit f)

Core Concepts

  • Automatic Differentiation - Forward and reverse mode AD
  • JIT Compilation - Just-in-time compilation for performance
  • Effect System - OCaml 5 algebraic effects for transformations
  • Composable Transformations - grad, jit, and other combinators
