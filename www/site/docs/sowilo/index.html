sowilo Documentation

This is the documentation section for sowilo, the computer vision library with JIT-compiled operations.

Getting Started

sowilo provides computer vision operations and image processing capabilities with GPU acceleration support.

Quick Example

open Sowilo
open Ndarray.Io

(* Load and process image *)
let img = imread "input.jpg"
let gray = Cv.rgb2gray img
let blurred = Cv.gaussian_blur gray ~sigma:2.0
let edges = Cv.canny blurred ~low:50. ~high:150.

(* Save result *)
let _ = imwrite edges "edges.jpg"

Core Concepts

  • Image I/O - Loading and saving various image formats
  • Filtering - Gaussian blur, median filter, and custom kernels
  • Edge Detection - Canny, Sobel, and other edge detectors
  • Morphological Operations - Erosion, dilation, opening, closing
