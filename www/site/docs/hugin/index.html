hugin Documentation

This is the documentation section for hugin, the publication-quality plotting library.

Getting Started

hugin provides 2D and 3D plotting capabilities with direct integration to ndarray data structures.

Quick Example

open Hugin
open Ndarray

(* Generate data *)
let x = Ndarray.linspace ~start:0. ~stop:10. 100
let y = Ndarray.sin x

(* Create plot *)
let fig = Figure.create ()
let ax = Figure.add_subplot fig
let _ = Axes.plot ax x y ~label:"sin(x)"
