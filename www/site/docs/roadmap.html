Key Technical Priorities

GPU Acceleration

First-class GPU support across the ecosystem with Metal (macOS) and CUDA backends.

rune: JIT compilation to GPU kernels
ndarray: Transparent GPU arrays
kaun: GPU-accelerated training

OCaml 5 Integration

Leverage algebraic effects for composable transformations and multicore parallelism.

Effects: Non-intrusive grad(), jit(), vmap()
Domains: Parallel array operations
Scheduler: Async I/O and computation

Type Safety Innovations

Push the boundaries of compile-time guarantees for numerical computing.

Shape Types: Compile-time dimension checking
Device Types: Memory location tracking
Grad Types: Differentiability guarantees
