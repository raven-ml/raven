kaun Documentation

This is the documentation section for kaun, the neural network library for deep learning.

Getting Started

kaun provides high-level APIs for building and training neural networks, built on top of rune's automatic differentiation.

Quick Example

open Kaun

(* Define a neural network *)
let model = Sequential.[
  Dense.create ~units:128 ~activation:Relu;
  Dropout.create ~rate:0.2;
  Dense.create ~units:10 ~activation:Softmax;
]

(* Compile and train *)
let compiled = Model.compile model 
  ~optimizer:(Adam.create ~lr:0.001)
  ~loss:CategoricalCrossentropy

Core Concepts

  • Layers - Dense, convolutional, dropout, and custom layers
  • Models - Sequential and functional model architectures
  • Training - Optimizers, loss functions, and training loops
  • Metrics - Evaluation and monitoring during training
