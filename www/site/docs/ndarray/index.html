ndarray Documentation

This is the documentation section for ndarray, the foundation array library of the Raven ecosystem.

Getting Started

ndarray provides N-dimensional arrays with linear algebra operations, similar to NumPy but with OCaml's type safety.

Quick Example

open Ndarray

(* Create a 3x3 matrix *)
let a = Ndarray.ones [3; 3] float32

(* Matrix operations *)
let b = Ndarray.random [3; 3] float32
let c = Ndarray.matmul a b

  • Indexing & Slicing - Access and modify array elements
  • Broadcasting - Operations between arrays of different shapes
