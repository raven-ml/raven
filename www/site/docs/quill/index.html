quill Documentation

This is the documentation section for quill, the interactive notebook environment for data science.

Getting Started

quill provides a Jupyter-like notebook environment with markdown-based notebooks and live OCaml code execution.

Quick Example

# Start Quill server
$ dune exec quill -- serve notebooks/

# Or execute a notebook
$ dune exec quill -- exec analysis.md

In a Notebook

```ocaml
open Ndarray
open Hugin

let data = Ndarray.random [100; 2] float32
let fig = Figure.create ()
let _ = Axes.scatter (Figure.add_subplot fig) data
```

Core Concepts

  • Notebook Format - Markdown-based notebooks for version control
  • Code Execution - Live OCaml evaluation with rich output
  • Plotting Integration - Seamless hugin plot display
  • Export Options - HTML, PDF, and command-line execution
