N-dimensional arrays with linear algebra operations. The foundation of the Raven ecosystem.

Core Features

Multi-dimensional Arrays

Efficient N-dimensional arrays with support for various data types and memory layouts.

High Performance

Zero-copy operations, optimized linear algebra, and efficient memory management with Bigarray.

NumPy Compatible

Familiar API design inspired by NumPy, making migration from Python straightforward.

Type Safety

Leverage OCaml's type system for compile-time guarantees and runtime safety.

Broadcasting

Automatic broadcasting for operations between arrays of different shapes and sizes.

Linear Algebra

Comprehensive linear algebra operations including matrix multiplication, decompositions, and solvers.

Quick Example

(* Create arrays and perform operations *)
open Ndarray

(* Create a 3x3 matrix *)
let a = Ndarray.ones [3; 3] float32

(* Create another matrix with random values *)
let b = Ndarray.random [3; 3] float32

(* Matrix multiplication *)
let c = Ndarray.matmul a b

(* Element-wise operations with broadcasting *)
let d = Ndarray.add c (Ndarray.scalar 2.0 float32)

(* Reductions and statistics *)
let mean = Ndarray.mean d
let sum = Ndarray.sum d ~axis:[0]

Extension Libraries

Specialized libraries that extend ndarray with additional functionality for I/O operations and dataset access.

ndarray-io

Integrated

Input/output operations for various data formats including images, NumPy files, and scientific data.

  • • Image formats (PNG, JPEG, TIFF)
  • • NumPy files (.npy, .npz)
  • • CSV and delimited text
  • • HDF5 scientific data
let img = Ndarray.Io.imread "image.png"
let _ = Ndarray.Io.save_npy arr "data.npy"

Integrated into the core ndarray library for seamless I/O operations.

ndarray-datasets

Production

Easy access to popular machine learning datasets with automatic downloading and preprocessing.

  • • MNIST handwritten digits
  • • CIFAR-10/100 image classification
  • • Iris, California Housing, Breast Cancer
  • • Automatic downloading and caching
let mnist = Datasets.Mnist.load ()
let x_train, y_train = mnist.train
