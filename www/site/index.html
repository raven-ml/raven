Raven

OCaml's Wings for Machine Learning

A comprehensive OCaml ecosystem for machine learning and data science, comparable to Python's NumPy/JAX/Matplotlib stack

From Python to OCaml

Python Stack

  • NumPy → Arrays & Linear Algebra
  • Matplotlib → Plotting & Visualization
  • JAX → Auto-diff & JIT
  • Jupyter → Interactive Notebooks

Raven Ecosystem

  • ndarray → N-dimensional arrays
  • hugin → Publication-quality plots
  • rune → Auto-diff & JIT compilation
  • kaun → Neural networks & deep learning
  • sowilo → Computer vision & image processing
  • quill → Interactive documents

Quick Start

# Install with opam
$ opam install ndarray hugin
# Or build from source
$ git clone https://github.com/your-org/raven.git
$ cd raven
$ dune build
