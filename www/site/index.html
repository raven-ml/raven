The Raven Ecosystem

Foundation ndarray

N-dimensional arrays with linear algebra. Start here - everything else builds on ndarray.

Like NumPy: array creation, indexing, broadcasting, linear algebra
Learn more →

Visualization hugin

Publication-quality 2D/3D plotting with direct ndarray integration.

Like Matplotlib: line plots, scatter, bar charts, 3D, subplots
Learn more →

ML/Compute rune

Automatic differentiation and JIT compilation for machine learning.

Like JAX: grad(), jit(), composable transformations
Learn more →

Deep Learning kaun

Neural network library with high-level APIs for deep learning.

Like Flax/PyTorch: layers, optimizers, training loops
Learn more →

Computer Vision sowilo

Computer vision library with JIT-compiled operations and GPU acceleration.

Like OpenCV/torchvision: image processing, object detection
Learn more →

Interactive quill

Notebook environment for interactive data science and documentation.

Like Jupyter: markdown-based notebooks, live code execution
Learn more →
