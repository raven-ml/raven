Ways to Contribute

For Users

Try the Libraries

Use Raven in your projects and share your experience. Your feedback shapes development priorities.

Report Issues

Found a bug or performance issue? Report it on GitHub with clear reproduction steps.

Share Examples

Create tutorials, blog posts, or example projects showing how you use Raven.

Improve Documentation

Help make our docs clearer by fixing typos, adding examples, or improving explanations.

For Developers

Core Development

Contribute to the core libraries: new features, optimizations, bug fixes, and API improvements.

Backend Development

Implement new backends: GPU acceleration (Metal, CUDA), distributed computing, or specialized hardware.

Performance Optimization

Profile and optimize performance-critical paths using SIMD, parallelization, and algorithmic improvements.

Domain Libraries

Build specialized libraries for computer vision, NLP, time series, or other domain-specific applications.

Community Guidelines

Code of Conduct

The Raven community is committed to providing a welcoming and inclusive environment for everyone, regardless of background, experience level, or identity.

We encourage:

  • • Respectful and constructive communication
  • • Helping newcomers learn and contribute
  • • Sharing knowledge and best practices
  • • Celebrating diverse perspectives and approaches
  • • Focusing on what's best for the community

We don't tolerate:

  • • Harassment or discriminatory behavior
  • • Personal attacks or inflammatory language
  • • Spam or off-topic discussions
  • • Sharing others' private information
  • • Any form of trolling or disruptive behavior

Issues can be reported to the maintainers privately. We're committed to addressing concerns promptly and fairly.
